# Pokedle
A Pokemon themed game based on Wordle. The user tries to guess a randomly chose Pokemon and is given info such as type(s), generation, and abilities that match with their guess.

# Instructions - Installing and Running
To run the program, ensure that you have installed requests by doing 

'pip install requests'

After requests is installed, run the program by doing 

'python pokedle.py'

# Instructions - Playing
Once the game has started, you will be prompted to enter the name of a Pokemon in the command line. Only valid Pokemon names (i.e bulbasaur, pikachu, charmander) are accepted. If your guess is incorrect, the program will reveal if the Pokemon you guessed had any common type(s), ability/abilities, or generation. If you give up, you can run ctrl+c to end the program before which the Pokemon's name will be revealed.
